          <div class="discuss-comment"><h1>SCSS / SASS Implementation</h1>

<h2>A more modern approach</h2>

<p>For those using SCSS (or easily converted to SASS), the following will be helpful. Effectively, make an element next to the checkbox, which is the one that you will style. When the checkbox is clicked, the CSS restyles the sister element (to your new, checked style). Code is below:</p>

<p></p><div class="snippet" data-lang="js" data-hide="false" data-console="true" data-babel="false">
<div class="snippet-code">
<pre class="snippet-code-css lang-css prettyprint-override"><code>label.checkbox {<font></font>
  input[type="checkbox"] {<font></font>
    visibility: hidden;<font></font>
    display: block;<font></font>
    height: 0;<font></font>
    width: 0;<font></font>
    position: absolute;<font></font>
    overflow: hidden;<font></font>
<font></font>
    &amp;:checked + span {<font></font>
      background: $accent;<font></font>
    }<font></font>
  }<font></font>
<font></font>
  span {<font></font>
    cursor: pointer;<font></font>
    height: 15px;<font></font>
    width: 15px;<font></font>
    border: 1px solid $accent;<font></font>
    border-radius: 2px;<font></font>
    display: inline-block;<font></font>
    transition: all 0.2s $interpol;<font></font>
  }<font></font>
}</code></pre>
<pre class="snippet-code-html lang-html prettyprint-override"><code>&lt;label class="checkbox"&gt;<font></font>
    &lt;input type="checkbox" /&gt;<font></font>
    &lt;span&gt;&lt;/span&gt;<font></font>
    Label text<font></font>
&lt;/label&gt;</code></pre>
</div>

          <div class="discuss-comment"><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这是仅CSS / HTML的版本，完全不需要jQuery或JavaScript，简单干净的HTML以及真正简单简短的CSS。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这是JSFiddle</font></font></p>

<p><a href="http://jsfiddle.net/v71kn3pr/" rel="nofollow noreferrer"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">http://jsfiddle.net/v71kn3pr/</font></font></a></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这是HTML：</font></font></p>

<pre><code>&lt;div id="myContainer"&gt;<font></font>
    &lt;input type="checkbox" name="myCheckbox" id="myCheckbox_01_item" value="red" /&gt;<font></font>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这是CSS</font></font></p>

<pre><code>#myContainer {<font></font>
    outline: black dashed 1px;<font></font>

}<font></font>
#myContainer input[type="checkbox"][name="myCheckbox"] {<font></font>
    display: none;<font></font>
}<font></font>
#myContainer input[type="checkbox"][name="myCheckbox"]:not(:checked) + label.box {<font></font>
    display: inline-block;<font></font>
    width: 25px;<font></font>
    height: 25px;<font></font>
    border: black solid 1px;<font></font>
    background: #FFF ;<font></font>
    margin: 5px 5px;<font></font>
}<font></font>
#myContainer input[type="checkbox"][name="myCheckbox"]:checked + label.box {<font></font>
    display: inline-block;<font></font>

    border: black solid 1px;<font></font>
    background: #F00;<font></font>
    margin: 5px 5px;<font></font>
}<font></font>

    vertical-align: top;<font></font>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这可以被适配为能够具有单独的单选框或复选框，复选框组以及单选按钮组。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">此html / css还可让您捕获标签上的点击，因此即使您仅单击标签，该复选框也将被选中和取消选中。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这种类型的复选框/单选按钮适用于任何形式，完全没有问题。</font><font style="vertical-align: inherit;">也已经使用PHP，ASP.NET（.aspx），</font></font><a href="https://en.wikipedia.org/wiki/JavaServer_Faces" rel="nofollow noreferrer"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">JavaServer Faces</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和</font></font><a href="http://en.wikipedia.org/wiki/Adobe_ColdFusion" rel="nofollow noreferrer"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ColdFusion进行</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">了测试。</font></font></p></div>
