<blockquote>
  <p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">警告：数组或迭代器中的每个子代都应具有唯一的“键”道具。</font></font></p>
</blockquote>

<p><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">注意：</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">人们经常犯的一个错误是</font></font><code>index</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在迭代时</font><font style="vertical-align: inherit;">将其</font><font style="vertical-align: inherit;">用作键</font></font><code>index</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，将元素用作键是一种反模式，您可以</font></font><a href="https://medium.com/@robinpokorny/index-as-a-key-is-an-anti-pattern-e0349aece318" rel="nofollow noreferrer"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在此处</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">了解更多信息</font><font style="vertical-align: inherit;">。</font><font style="vertical-align: inherit;">简而言之，如果</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">不是</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">静态列表，则永远不要将其</font></font><code>index</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">用作键。</font></font></p>

<hr>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">现在，在</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ObjectRow</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">组件中，您可以从其属性访问该对象。</font></font></p>

<p><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">内部ObjectRow组件</font></font></strong></p>

<p><code>const { object } = this.props</code></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">要么 </font></font></p>

<p><code>const object = this.props.object</code></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这应该获取您从父组件传递到</font><strong><font style="vertical-align: inherit;">ObjectRow</font></strong><font style="vertical-align: inherit;">组件中</font><font style="vertical-align: inherit;">的变量</font></font><code>object</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">的</font></font><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">对象</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font><font style="vertical-align: inherit;">现在，您可以根据自己的目的吐出该对象中的值。</font></font></p>
