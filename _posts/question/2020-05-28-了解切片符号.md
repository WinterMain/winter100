          <div class="discuss-comment"><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">基本切片技术是定义起点，终点和步长-也称为步幅。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">首先，我们将创建一个值列表以用于切片。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">创建两个要切片的列表。</font><font style="vertical-align: inherit;">第一个是从1到9的数字列表（列表A）。</font><font style="vertical-align: inherit;">第二个也是一个数字列表，从0到9（列表B）：</font></font></p>

<pre class="lang-py prettyprint prettyprinted" style=""><code><span class="pln">A </span><span class="pun">=</span><span class="pln"> list</span><span class="pun">(</span><span class="pln">range</span><span class="pun">(</span><span class="lit">1</span><span class="pun">,</span><span class="pln"> </span><span class="lit">10</span><span class="pun">,</span><span class="pln"> </span><span class="lit">1</span><span class="pun">))</span><span class="pln"> </span><span class="com"># Start, stop, and step</span><span class="pln">

<p><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">基本切片</font></font></strong></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">用于切片的扩展索引语法为aList [start：stop：step]。</font><font style="vertical-align: inherit;">start参数和step参数都默认为none-唯一需要的参数是stop。</font><font style="vertical-align: inherit;">您是否注意到这类似于使用范围定义列表A和B的方式？</font><font style="vertical-align: inherit;">这是因为slice对象代表由range（开始，停止，步进）指定的索引集。</font><font style="vertical-align: inherit;">Python 3.4文档。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如您所见，仅定义stop将返回一个元素。</font><font style="vertical-align: inherit;">由于开始默认为无，因此这意味着只检索一个元素。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">重要的是要注意，第一个元素是索引0，</font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">而不是</font></font></em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">索引1。这就是为什么我们在此练习中使用2个列表的原因。</font><font style="vertical-align: inherit;">列表A的元素根据顺序位置编号（第一个元素为1，第二个元素为2，依此类推），而列表B的元素为将用于为其编号的数字（第一个元素为[0]，第一个元素为[0]，等等。）。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">使用扩展的索引语法，我们检索值的范围。</font><font style="vertical-align: inherit;">例如，所有值都用冒号检索。</font></font></p>

<pre class="lang-py prettyprint prettyprinted" style=""><code><span class="pln">A</span><span class="pun">[:]</span></code></pre>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">要检索元素的子集，需要定义开始位置和停止位置。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">给定模式aList [start：stop]，从列表A中检索前两个元素。</font></font></p></div>

          <div class="discuss-comment"><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">我认为，如果以以下方式（继续阅读）看待它，您将更好地理解和记住Python字符串切片表示法。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">让我们使用以下字符串...</font></font></p>

<pre class="lang-py prettyprint prettyprinted" style=""><code><span class="pln">azString </span><span class="pun">=</span><span class="pln"> </span><span class="str">"abcdefghijklmnopqrstuvwxyz"</span></code></pre>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">对于那些不知道的人，您可以</font></font><code>azString</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">使用符号</font><font style="vertical-align: inherit;">来创建任何子字符串</font></font><code>azString[x:y]</code></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">来自其他编程语言的那是常识受到损害的时候。</font><font style="vertical-align: inherit;">x和y是什么？</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在寻求一种记忆技术时，我不得不坐下来并运行几种方案，该技术将帮助我记住x和y是什么，并帮助我在第一次尝试中正确地分割字符串。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">我的结论是，x和y应该被视为包围我们要附加的字符串的边界索引。</font><font style="vertical-align: inherit;">因此，我们应该将表达式视为</font></font><code>azString[index1, index2]</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">或什至更清晰</font></font><code>azString[index_of_first_character, index_after_the_last_character]</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这是该示例的可视化示例...</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">因此，您要做的就是将index1和index2设置为所需子字符串周围的值。</font><font style="vertical-align: inherit;">例如，要获取子字符串“ cdefgh”，可以使用</font></font><code>azString[2:8]</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，因为“ c”左侧的索引是2，而在“ h”右侧的索引是8。</font></font></p>

<h3>Relation to <code>slice()</code> object</h3>

<p>The slicing operator <code>[]</code> is actually being used in the above code with a <code>slice()</code> object using the <code>:</code> notation (which is only valid within <code>[]</code>), i.e.:</p>

<pre class="lang-py prettyprint prettyprinted" style=""><code><span class="pln">a</span><span class="pun">[</span><span class="pln">start</span><span class="pun">:</span><span class="pln">stop</span><span class="pun">:</span><span class="pln">step</span><span class="pun">]</span></code></pre>

<p>is equivalent to:</p>

<pre class="lang-py prettyprint prettyprinted" style=""><code><span class="pln">a</span><span class="pun">[</span><span class="pln">slice</span><span class="pun">(</span><span class="pln">start</span><span class="pun">,</span><span class="pln"> stop</span><span class="pun">,</span><span class="pln"> step</span><span class="pun">)]</span></code></pre>

<p>Slice objects also behave slightly differently depending on the number of arguments, similarly to <code>range()</code>, i.e. both <code>slice(stop)</code> and <code>slice(start, stop[, step])</code> are supported.
To skip specifying a given argument, one might use <code>None</code>, so that e.g. <code>a[start:]</code> is equivalent to <code>a[slice(start, None)]</code> or <code>a[::-1]</code> is equivalent to <code>a[slice(None, None, -1)]</code>.</p>

<p>While the <code>:</code>-based notation is very helpful for simple slicing, the explicit use of <code>slice()</code> objects simplifies the programmatic generation of slicing.</p></div>
