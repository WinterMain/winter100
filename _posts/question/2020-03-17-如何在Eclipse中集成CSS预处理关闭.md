          <div class="discuss-comment"><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">我只是想出了如何在Eclipse中执行此操作。</font><font style="vertical-align: inherit;">我承认此解决方案不支持100％SASS，使用嵌套CSS时颜色会有点时髦，但是比查看纯文本要好得多，并且您不需要安装单独的编辑器。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">您需要将.scss文件类型与Eclipse中的本机Eclipse CSS编辑器关联起来[第1部分]。</font><font style="vertical-align: inherit;">完成之后，您还需要将.scss文件类型添加到本机CSS编辑器中，以便CSS编辑器能够打开它[第2部分]。</font><font style="vertical-align: inherit;">以下是Eclipse的步骤（我正在为Web开发人员运行Eclipse Java EE IDE，Indigo）：</font></font></p>

<h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">第1部分-将.scss文件类型与本机Eclipse CSS编辑器相关联</font></font></h3>

<ol>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">去 </font></font><code>Window &gt; Preferences</code></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">深入到 </font></font><code>General &gt; Editors &gt; File Associations</code></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在“文件关联”窗格中，单击右上角的“添加...”按钮。</font></font></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">对于</font></font><code>File Type:</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，输入</font></font><code>*.scss</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，然后单击确定</font></font><code>OK</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p></li>
<li><p><font style="vertical-align: inherit;"></font><code>*.scss</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在</font></font><code>File Associations</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">列表中</font><font style="vertical-align: inherit;">找到</font><font style="vertical-align: inherit;">条目</font><font style="vertical-align: inherit;">并将其选中。</font></font></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在</font></font><code>*.scss</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">底部窗格上</font><font style="vertical-align: inherit;">选择之后</font></font><code>Associated editors:</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，单击</font></font><code>Add...</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">按钮。</font></font></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">确保</font></font><code>Internal editors</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在顶部被选中，然后找到并选择</font></font><code>CSS Editor</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，然后单击</font></font><code>OK</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p></li>
</ol>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这将文件类型.scss与eclipses本机CSS编辑器关联。</font><font style="vertical-align: inherit;">现在，我们必须配置本机CSS编辑器以支持.scss文件。</font><font style="vertical-align: inherit;">为此，请按照下列步骤操作：</font></font></p>

<h3><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">第2部分-将.scss文件类型添加到本机CSS编辑器</font></font></h3>

<ol>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">去 </font></font><code>Window &gt; Preferences</code></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">深入到 </font></font><code>General &gt; Content Types</code></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在“内容类型”窗格中，展开</font></font><code>Text</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，然后选择</font></font><code>CSS</code></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">之后</font></font><code>CSS</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">被选中，在底部</font></font><code>File associations:</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">窗格中，单击</font></font><code>Add...</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">按钮。</font></font></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">对于</font></font><code>Content type:</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，输入</font></font><code>*.scss</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，然后单击确定</font></font><code>OK</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p></li>
<li><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">单击</font></font><code>OK</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">以关闭“首选项”窗口。</font></font></p></li>
</ol>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">全做完了。</font><font style="vertical-align: inherit;">现在您需要做的就是关闭所有打开的.scss文件，然后重新打开它们，并在Eclipse中打开.scss文件中的wha-la，css颜色！</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">注意：如果未显示css颜色，则可能必须执行以下操作：</font></font><code>Right click the .scss file &gt; Open With &gt; CSS Editor</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">希望这可以帮助。 </font></font></p></div>
