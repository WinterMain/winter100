    <div class="article-content"><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">我正在一个Web应用程序上工作，我希望其中的内容能填满整个屏幕的高度。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">该页面具有标题，其中包含徽标和帐户信息。</font><font style="vertical-align: inherit;">这可以是任意高度。</font><font style="vertical-align: inherit;">我希望内容div将页面的其余部分填充到底部。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">我有一个标题</font></font><code>div</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和一个内容</font></font><code>div</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font><font style="vertical-align: inherit;">目前，我正在使用表格进行布局，如下所示：</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CSS和HTML</font></font></p>

<p></p><div class="snippet" data-lang="js" data-hide="false" data-console="false" data-babel="false">
<div class="snippet-code">
<pre class="snippet-code-css lang-css prettyprint-override"><code>#page {<font></font>
    height: 100%; width: 100%<font></font>
}<font></font>
<font></font>
#tdcontent {<font></font>
    height: 100%;<font></font>
}<font></font>
<font></font>
#content {<font></font>
    overflow: auto; /* or overflow: hidden; */<font></font>
}</code></pre>
<pre class="snippet-code-html lang-html prettyprint-override"><code>&lt;table id="page"&gt;<font></font>
    &lt;tr&gt;<font></font>
        &lt;td id="tdheader"&gt;<font></font>
            &lt;div id="header"&gt;...&lt;/div&gt;<font></font>
        &lt;/td&gt;<font></font>
    &lt;/tr&gt;<font></font>
    &lt;tr&gt;<font></font>
        &lt;td id="tdcontent"&gt;<font></font>
            &lt;div id="content"&gt;...&lt;/div&gt;<font></font>
        &lt;/td&gt;<font></font>
    &lt;/tr&gt;<font></font>
&lt;/table&gt;</code></pre>
</div>
</div>
<p></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">页面的整个高度已填满，不需要滚动。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">对于content div中的任何内容，设置</font></font><code>top: 0;</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">将其放在标题的正下方。</font><font style="vertical-align: inherit;">有时，内容将是一个实际表，其高度设置为100％。</font><font style="vertical-align: inherit;">把</font></font><code>header</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">里面</font></font><code>content</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">不会让这种工作。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">是否有一种无需使用即可达到相同效果的方法</font></font><code>table</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">？</font></font></p>

<p><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">更新：</font></font></strong></p>

<p>Elements inside the content <code>div</code> will have heights set to percentages as well. So something at 100% inside the <code>div</code> will fill it to the bottom. As will two elements at 50%.</p>

<p><strong>Update 2:</strong></p>

<p>For instance, if the header takes up 20% of the screen's height, a table specified at 50% inside <code>#content</code> would take up 40% of the screen space. So far, wrapping the entire thing in a table is the only thing that works.</p></div>

          <div class="discuss-comment"><p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">分拆外星人的想法...</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">与启用CSS3的浏览器流行的flex box相比，这似乎是一种更干净的解决方案。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">只需对内容块使用带有calc（）的min-height（而不是height）。 </font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">calc（）以100％开头，并减去页眉和页脚的高度（需要包括填充值）</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">使用“最小高度”代替“高度”特别有用，因此它可以与javascript呈现的内容和Angular2之类的JS框架一起使用。</font><font style="vertical-align: inherit;">否则，一旦显示了javascript呈现的内容，该计算就不会将页脚推到页面的底部。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这是使用50px高度和20px填充的页眉和页脚的简单示例。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">HTML：</font></font></p>

<pre><code>&lt;body&gt;<font></font>
    &lt;header&gt;&lt;/header&gt;<font></font>
    &lt;div class="content"&gt;&lt;/div&gt;<font></font>
    &lt;footer&gt;&lt;/footer&gt;<font></font>
&lt;/body&gt;<font></font>
</code></pre>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CSS：</font></font></p>

<pre><code>.content {<font></font>
    min-height: calc(100% - (50px + 20px + 20px + 50px + 20px + 20px));<font></font>
}<font></font>
</code></pre>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">假设我们具有</font></font><em><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">固定的高度</font></font></em> <code>header</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和</font></font><code>footer</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">元素，并且我们希望</font></font><code>section</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">标签采用整个可用的垂直高度...</font></font></p>

}<font></font>
