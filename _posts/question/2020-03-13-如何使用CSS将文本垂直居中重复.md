          <div class="discuss-comment"><p><strong>Absolute Positioning and Stretching</strong></p>

<p>As with the method above this one begins by setting positioning on the parent and child elements as relative and absolute respectively. From there things differ.</p>

<p>In the code below I’ve once again used this method to center the child both horizontally and vertically, though you can use the method for vertical centering only.</p>

<p><strong>HTML</strong></p>

<pre><code>&lt;div id="parent"&gt;<font></font>
    &lt;div id="child"&gt;Content here&lt;/div&gt;<font></font>
&lt;/div&gt;<font></font>
</code></pre>

<p><strong>CSS</strong></p>

<pre><code>#parent {position: relative;}<font></font>
#child {<font></font>
    position: absolute;<font></font>
    top: 0;<font></font>
    bottom: 0;<font></font>
    left: 0;<font></font>
    right: 0;<font></font>
    width: 50%;<font></font>
    height: 30%;<font></font>
    margin: auto;<font></font>
}<font></font>
</code></pre>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">使用此方法的想法是，通过将上，下，右和左值设置为0，尝试使子元素延伸到所有四个边缘。由于我们的子元素小于父元素，因此无法到达所有子元素四个边缘。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">但是，将auto设置为所有四个边的边距会使相反的边距相等，并在父div的中心显示子div。</font></font></p>

<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">不幸的是，上面的方法在Internet Explorer 7及更低版本中不起作用，并且像以前的方法一样，子div中的内容可能会变得太大，导致其被隐藏。</font></font></p></div>

<p>With just three lines of CSS (excluding vendor prefixes) we can do it with the help of a transform: translateY vertically centers whatever we want, even if we don’t know its height.</p>

<p>The CSS property transform is usually used for rotating and scaling elements, but with its translateY function we can now vertically align elements. Usually this must be done with absolute positioning or setting line-heights, but these require you to either know the height of the element or only works on single-line text, etc.</p>

<p>So, to do this we write:</p>

<pre><code>.element {<font></font>
    position: relative;<font></font>
    top: 50%;<font></font>
    transform: translateY(-50%);<font></font>
} <font></font>
</code></pre>
