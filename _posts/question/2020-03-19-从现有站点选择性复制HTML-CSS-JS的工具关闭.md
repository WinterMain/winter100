<p><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">像大多数Web开发人员一样，我偶尔也喜欢查看网站的来源，以了解其标记的构建方式。</font><font style="vertical-align: inherit;">Firebug和Chrome开发者工具之类的工具使检查代码变得容易，但是如果我要复制一个孤立的部分并在本地使用它，则复制所有单个元素及其关联的CSS会很痛苦。</font><font style="vertical-align: inherit;">保存整个源代码并剪切不相关的代码可能需要做很多工作。</font></font></p>

<pre><code>var el = document.querySelector("#someid");<font></font>

<font></font>

<font></font>
    els[i].setAttribute("style", window.getComputedStyle(els[i]).cssText);<font></font>
<font></font>
}<font></font>
</code></pre>
