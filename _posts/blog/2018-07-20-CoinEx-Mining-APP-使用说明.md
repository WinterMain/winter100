    <div class="article-content"><p>一款用于CoinEx交易挖矿的APP，一键挖矿，可视化挖矿，无需托管服务，只需一个浏览器，便可轻松运行&hellip;&nbsp;<br />
An Web APP for CoinEx mining. Beaty UI, easy to operate. What you need is only a browser&hellip;</p>

<h2>使用说明</h2>

<ul>
	<li><strong>本APP是WebAPP，直接点开文件，便可在浏览器中运行</strong></li>
	<li><strong>浏览器需要跨域处理，关于跨域请访问</strong><a href="https://www.samyoc.com/single/86" target="_blank">浏览器跨域指导</a></li>
	<li><strong>请填写交易对，交易额度，风险控制栏目，再开始</strong></li>
	<li><strong>APP供爱好CoinEx者交流使用</strong></li>
	<li><strong>建议在不清楚的情况下关闭API提现，用的放心</strong></li>
</ul>

<h2>交易对</h2>

<ul>
	<li>买卖货币：交易的货币比如买卖CET等等（填大写）</li>
	<li>定价货币：比如USDT， BTC市场中则是BTC（填大写）</li>
</ul>

<h2>交易额度</h2>

<ul>
	<li>
	<p>每笔交易额：APP实现挖矿的原理即是同时下买单和卖单完成一次挖矿，每笔交易额便是设置每笔单的金额，比如我选择了BTC/USDT交易对，设置了每笔交易额是50USDT，那每次的下的买单和买单的金额将等值于50USDT。</p>
	</li>
	<li>
	<p>最小交易额度：当买卖的货币或者定价的货币的的金额不足最小额度时，APP将以市价买入或卖出对应的货币，以达到买卖货币和定价货币等值，保证资金利用率。比如设置最小交易额度为5，在某一次交易中BTC价值不足5USDT时，这是后便会花一半的USDT市价买入BTC。</p>
	</li>
</ul>

<h2>风险控制</h2>

<ul>
	<li>
	<p>挖矿设置：建议设置为80%。当挖矿挖出的数量达到80%时比较适合停止。</p>
	</li>
	<li>
	<p>设置了每秒卖出多少个CET后，程序会自动帮你在每隔多少秒之后<strong>市价</strong>卖出CET，如果不想程序自动卖，你设置为0就可以了。</p>
	</li>
</ul>
